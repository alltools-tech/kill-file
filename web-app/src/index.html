Help & FAQ
  • Q: Which files can I convert?
    A: PDF, images (JPG, PNG, SVG, etc), Office files (DOCX, XLSX, PPTX, TXT, etc).
  • Q: Maximum file size?
    A: 12 MB per file.
  • Q: How to bulk download converted files?
    A: Use the "Download All as ZIP" button after conversion.
  • Q: Which languages are supported for OCR?
    A: English, Hindi, Urdu, Chinese, Japanese, Arabic, Russian, French.
  • Q: My conversion fails! What should I do?
    A: Check file type/size, refresh page, or contact support.
